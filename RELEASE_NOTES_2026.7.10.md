# Time Library 2026.7.10

2026.7.10 is a maintenance update over 2026.7.7.2. It improves the accuracy of
Reading Room information, library counts, local service status, and watcher
status reporting in the local console while keeping existing compatibility
behavior.

## What Changed

- Reading Room project pages now use the local source-backed catalog projection.
- Library counts and local service health are displayed more consistently.
- Watcher status reporting is more reliable on supported Windows installations.

## 中文

2026.7.10 是 2026.7.7.2 之上的维护更新，优化阅读室信息、馆藏计数、
本机服务状态和监听状态显示，同时保持已有兼容行为。

### 本版本包含

- 阅读室项目页改用本机可回源目录投影。
- 馆藏计数和本机服务健康状态显示更一致。
- 优化受支持 Windows 安装上的监听状态显示。
